## 📌 Overview

**Gestion_Bibliotheque** is a comprehensive library management system designed to help librarians and users manage books, members, loans, and returns efficiently through a modern Java Swing interface.

### 🔍 Key Features

- 👤 **Member Management:** Add, modify, delete, and search library members
- 📚 **Book Management:** Add, modify, delete, and search books
- 🔄 **Loan Management:** Borrow and return books, track current and past loans
- 📊 **Statistics:** View most borrowed books, active users, and popular genres
- 🔒 **User Authentication:** Secure access for administrators
- 🖥️ **Modern UI:** Clean, user-friendly interface built with Swing

## 🚀 Getting Started

### Prerequisites

- Java 11 or higher
- Maven

---

### Installation

Clone the repository:
```bash
git clone https://github.com/amineboubekri/Gestion_Bibliotheque.git
cd Gestion_Bibliotheque
```

Build the project with Maven:
```bash
mvn clean install
```

---

### ▶️ Usage

Run the application:
```bash
mvn exec:java -Dexec.mainClass="view.BiblioView"
```

Or run the generated JAR (after building):
```bash
java -jar target/library-management-1.0-SNAPSHOT.jar
```

---

### ✅ Features

- **Member Management:**
  - Register, update, and remove members
  - Search and filter members
- **Book Management:**
  - Add, update, and remove books
  - Search and filter books
- **Loan Management:**
  - Borrow and return books
  - Track loan history
- **Statistics:**
  - Most borrowed books
  - Most active users
  - Most popular genres
